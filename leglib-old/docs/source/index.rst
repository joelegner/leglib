Other Requirements
==================

1. `Python 2.7 <http://www.python.org/download/releases/2.7/>`_.

#. `GCC - The GNU Compiler Collection <http://gcc.gnu.org/>`_ for compiling and installing Numpy.

#. `GNU Make <https://www.gnu.org/software/make/>`_.

Installation
============

1. Verify :ref:`other-requirements` are met.  Install anything missing.

2. Type at a bash prompt:

.. code-block:: bash

   $ svn co https://svn.ursavus.com/leglib leglib
   $ cd leglib
   $ pip install -r requirements.txt

Next see :ref:`executing-tests`.
